description=[h1][FZ] FrameworkZ[/h1]
description=FrameworkZ is an open-source modding framework for Project Zomboid Build 41 (with plans for B42 compatibility). Designed to help modders and server owners build cleaner, more organized, and more expandable gameplay systems.
description=
description=It is not a content-heavy gameplay mod by itself. Instead, FrameworkZ provides the foundation for other mods, roleplay systems, server tools, interfaces, and shared functionality to be built on top of.
